## 功能检查

- [ ] 从绿色目录的 `bin/UTMS.exe` 启动，初始 UDP 状态为红色且端口为 `10000`。
- [ ] 启动监听后状态变黄；模拟器发送合法帧后变绿；停止后恢复红色。
- [ ] 在线地图能显示雷达和五类目标；Key 缺失或无效时显示明确错误。
- [ ] 在线街道图与卫星图可切换。
- [ ] 离线地图能读取 `bin/data/map/amap/{z}/{x}/{y}.png`；瓦片缺失时显示占位内容且不连续弹窗。
- [ ] 在线/离线模式切换后保持中心、缩放和选中目标。
- [ ] 点击表格目标后地图居中并高亮；点击地图目标后表格选中并滚动到对应行。
- [ ] 表格类型筛选、航迹 ID/速度/距离/时间排序和缺失值末尾规则正确。
- [ ] 柱状图、饼图和底部五类数量随当前帧同步更新，自行车计入饼图车辆类。
- [ ] 分割器可拖动，窗口缩放后地图、表格和图表正常自适应。
- [ ] 非法 JSON 保留上一帧；重复包和乱序包不使界面回退。
- [ ] `bin/logs/utms.log` 记录规定事件，轮转后最多保留 5 个日志文件。

## 范围检查

- [ ] 界面和数据模型不保存历史点迹或轨迹线。
- [ ] 仅处理单雷达当前帧快照，不做多雷达融合。
- [ ] 应用不执行 WGS-84、BD-09 或其他坐标转换。
- [ ] 不持久化端口、地图、窗口或表格配置。
- [ ] 发布目录不包含真实 `amap.json`、FFmpeg、ONNX Runtime、YOLO 模型或正式安装包。

所有项目勾选且无未解决阻断问题后，方可将该版本标记为通过人工验收。
